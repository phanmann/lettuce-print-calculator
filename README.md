# 🏷️ Lettuce Print - Professional Label Pricing Calculator

A comprehensive web-based pricing calculator for custom label printing with volume discounts and professional quoting capabilities.

## ✨ Features

### 🎯 Core Functionality
- **Custom Label Dimensions**: Width × length input with real-time area calculation
- **Material Options**: Standard and BOPP facestock with premium pricing
- **Laminate Selection**: Matte and Gloss finishing options
- **Actual Ink Cost Input**: Use real test print data for maximum accuracy
- **Volume Pricing**: Research-backed discount tiers (3%, 7%, 12%, 18%)

### 💰 Professional Pricing
- **Dual Display**: Your cost vs suggested sell price (100% markup)
- **Volume Discounts**: Automatic application at 1K, 5K, 10K, 25K+ units
- **6-Tier Breakdown**: Pricing table at key quantity levels
- **Cost Transparency**: Ink + Facestock + Laminate breakdown

### 📊 Sales Tools
- **Real-time Calculations**: Updates as you type
- **Export Functionality**: Professional quote downloads
- **Print Optimization**: Clean layouts for customer presentations
- **Material Comparison**: Side-by-side Standard vs BOPP pricing

## 📋 How to Use

### Step 1: Label Specifications
- Enter **Width** and **Length** in inches (decimal allowed)
- Select **Material**: Standard or BOPP
- Choose **Laminate**: Matte or Gloss finish

### Step 2: Actual Ink Cost
- Run a test print on your physical printer
- Calculate ink cost per label from actual usage
- Enter precise value in "Actual Ink Cost per Label" field

### Step 3: Quantity & Pricing
- Enter desired quantity
- Review automatic volume discount application
- See cost breakdown and suggested sell price

### Step 4: Generate Quotes
- Use "Print Quote" for customer presentations
- Export professional quote files
- Reference 6-tier pricing breakdown for sales conversations
